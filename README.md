# Dishcovery: A Restaurant/Recipe Finder App
Personalized Food Discovery Tool

A web application that suggests nearby restaurants and cooking recipes to users, using Google Maps API and Custom Google Search.

## Usage
To use this app, go to `nikki2906.pythonanywhere.com/`. You can also install and run the app locally by running:
```
git clone https://github.com/nikki2906/Dishcovery.git
python3 app.py
```

## Features
- [x] Restaurant recommendations using Google Maps API
- [x] Sentiment analysis of customer reviews for value, ambience, and service
- [x] Recipe suggestions via embedded Google Search Engine
- [x] Customized results based on the user's location, cuisine preference, and budget
- [x] User-friendly interface for easy input and browsing
